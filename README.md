<h1>Escopo</h1>
Software de Gestão de biblioteca Web 
<ul>
  <li><Cadastro completo dos livros (nome, autor, categoria, ISBN, Quantidade)/li>
  <li>Controle de estoque (quantidade disponível e quantidade emprestada)/li>
  <li>Status do livro (esgotado, estoque baixo e Disponível)</li>
  <li>Controle de Empréstimos (registrar saída, registrar devolução, extenção de prazo e lista de livros/pessoas atrasados)</li>
  <li>Relatórios (total de livros, livros mais procurados e controle de caixa)</li>
</ul>
